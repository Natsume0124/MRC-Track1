Track-Specific Requirements

Track 1: Visual Grounding (VG-RS)

  • Model Submission: Submit multimodal reasoning models with PyTorch/TensorFlow compatibility for bounding box prediction tasks.
  • Documentation: Include technical documentation explaining scene perception mechanisms and spatial reasoning implementation. Must annotate data preprocessing workflows [[WorkShop]].
  • Result Format: JSON file with structure: 
    [{
  "image_path": "images\\*.jpg",
  "question": "Target description",
  "result": [[x1,y1],[x2,y2]]  // Top-left/bottom-right coordinates
}]
